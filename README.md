## Installation

Open a terminal and clone the repository :

`git clone https://github.com/jbbrejon/openclassrooms-dajs-p12.git`

### Branch

main

### NPM

From the root folder of the project, open a terminal and run:

`npm install`


## Start Application

From the root folder of the project, open a terminal and run: 

`npm start`

Open a browser and go to 

`http://localhost:3001`
